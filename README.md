[PHPoole](http://github.com/Narno/PHPoole/) Gallery plugin.

Installation
------------

Add this to your PHPoole ```composer.json``` file:
```
{
  "require": {
    "PHPoole/Plugin-Gallery": "master-dev"
  },
  "repositories": [
    {
      "type": "vcs",
      "url": "https://github.com/PHPoole/Plugin-Gallery"
    }
  ]
}
```
